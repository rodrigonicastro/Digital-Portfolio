CS2 POV Finder

Summer 2026
Python TypeScript PostgreSQL Alembic Azure

Website to help CS2 enthusiasts find POV videos of specific players or roles. These videos are scattered across multiple YouTube channels, and there was no easy way to search for them, you had to manually look at each channel one by one. This website fixes this by weekly running a scraper on a leaderboard with players’ roles information, and matching players with videos pulled hourly with the YouTube API. Users subscribe to maps, roles, and players, and select what they want to watch and if they want to get notified or not.

Connect 4 - OOAD

Spring 2026
Java SwingUI OOP

Developed a Connect4 game using Java, and displayed using the SwingUI interface. This is basically the 2.0 version of the Connect4 game I developed for the Embedded Systems class. Many OOP design patterns and concepts were implemented in the coding of the game, specially to improve the Player, Board, and Connect4 classes. The game has a single-player mode (that can be played against random moves or AI), and a two-player mode.

Connect 4 - Embedded Software Engineering

Spring 2025
C STM32 HAL Embedded Systems

This is the final project for my Embedded Software Engineering class. Using a STM32 board and the peripherals in it, I developed a game of Connect 4 using the touch screen and buttons. The application is fully coded in C, with help of the HAL (Hardware Abstraction Layer) to configure and initialize all of the needed peripherals.
